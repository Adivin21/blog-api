# Blog API

API RESTful pour la gestion d'articles de blog.

##  Technologies

- **Node.js** - Runtime JavaScript
- **Express.js** - Framework web
- **SQLite3** - Base de données légère
- **Swagger** - Documentation API

##  Installation

```bash
# Cloner le dépôt
git clone https://github.com/Adivin21/blog-api.git

cd blog-api

npm init -y
# Installer les dépendances
# Installe les paquets nécessaires :

npm install express sqlite3 swagger-jsdoc swagger-ui-express cors
npm install --save-dev nodemon

npm install

npm run dev
